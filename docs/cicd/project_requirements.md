# Project Requirements (Week2 Day5)

## Functional
- Automated deployment via GitHub Actions
- Multi-environment (dev/test/prod)
- AI agent backend + frontend deployment

## Non-Functional
- Secure (no long-lived credentials)
- Reproducible infra (Terraform)
- Scalable (serverless)

## DevOps
- CI/CD pipeline
- Remote state (S3 + DynamoDB)
- OIDC authentication

## Success Criteria
- git push → deploy
- manual trigger → env deploy
- full teardown capability
